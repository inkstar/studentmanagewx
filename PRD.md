# 学生管理小程序 PRD

## 1. 文档信息
- 版本：v1.0
- 日期：2026-03-03
- 状态：MVP 需求基线
- 负责人：产品/研发协作

## 2. 项目背景
教培机构老师日常需要记录学生信息、课堂表现、考试成绩，并持续追踪学生薄弱点。现有纸质或分散表格方式导致数据不统一、回溯困难、复盘效率低。

## 3. 目标与范围
### 3.1 产品目标
1. 帮助老师高效完成学生信息管理与课堂记录。
2. 形成可追踪的成绩与薄弱点数据，支持教学复盘。
3. 为机构管理员提供班级、课程、科目和标签治理能力。

### 3.2 MVP 范围
1. 学生基本信息管理。
2. 每次上课情况登记（小班课/1对多）。
3. 考试成绩记录（总分+科目分+评语）。
4. 薄弱点手动标记与统计分析。
5. 管理端基础配置（班级、课程、科目、老师绑定）。
6. CSV 导入、CSV/PDF 导出。

### 3.3 非范围
1. 家长端。
2. 题目级自动诊断。
3. 多租户 SaaS 化。

## 4. 用户角色与权限
### 4.1 管理员
1. 管理全机构班级、课程、科目、标签。
2. 绑定老师与班级。
3. 执行批量导入与数据导出。

### 4.2 老师
1. 仅查看并编辑自己负责班级的数据。
2. 维护课堂记录、考试记录、薄弱点标记。

## 5. 核心功能需求
### 5.1 学生基本信息
- 字段：姓名、性别、年级、学校、联系方式、家长信息、班级归属、备注。
- 能力：新增、编辑、归档、搜索、按班级筛选。

### 5.2 上课情况登记
- 按课程实例创建课堂记录。
- 对班级内每位学生登记：出勤、课堂表现、作业、评语。
- 支持批量提交与回显。

### 5.3 考试成绩记录
- 记录考试名称、日期、班级。
- 录入学生总分、科目分、评语。
- 支持按学生查看历史成绩趋势。

### 5.4 薄弱点分析与记录
- 老师可为学生手动打薄弱点标签。
- 标签支持老师创建与管理员全局维护。
- 提供学生维度、班级维度的高频薄弱点统计。

### 5.5 导入导出
- CSV 导入学生数据（模板校验、错误行反馈）。
- CSV 导出原始记录。
- PDF 导出学生阶段报告（基础信息、成绩趋势、薄弱点汇总）。

## 6. 业务流程
1. 管理员完成班级/科目/课程/老师配置。
2. 老师维护学生档案并开展课堂登记。
3. 考后老师录入成绩并标记薄弱点。
4. 系统生成统计视图，支持导出报告。

## 7. 数据模型（概要）
1. users：账号、角色、班级权限。
2. classes：班级信息与老师关联。
3. subjects：机构自定义科目。
4. students：学生档案。
5. lessons / lesson_records：课程与到课表现。
6. exams / exam_scores：考试与成绩。
7. weakness_tags / weakness_logs：标签与打标日志。

## 8. 非功能需求
1. 安全：云函数校验权限，不信任前端传参。
2. 性能：常用列表分页，首页关键查询可在 2 秒内返回。
3. 可用性：支持弱网重试与幂等提交。
4. 审计：关键操作可追踪操作人和时间。

## 9. 验收标准
1. 老师可在 3 分钟内完成一节课全班记录。
2. 学生详情可查看课堂与成绩历史。
3. 薄弱点可按学生与班级统计。
4. 管理员可完成 CSV 批量导入并获取错误报告。
5. 系统可导出 CSV 与 PDF 报告。

## 10. 里程碑建议
1. 第 1 周：基础配置与权限。
2. 第 2 周：学生管理与导入。
3. 第 3 周：课堂记录与成绩。
4. 第 4 周：薄弱点分析、导出、联调验收。
